# Voxiom Userscripts

Colección personal de userscripts para Tampermonkey en Voxiom.io.

Scripts JavaScript diseñados para personalizar la experiencia visual y de jugabilidad en el navegador.  
**Uso exclusivamente educativo y experimental.**

> **Advertencia**: Estos scripts modifican el cliente del juego y pueden violar los Términos de Servicio de Voxiom.io.  
> Úsalos bajo tu propia responsabilidad y preferentemente en cuentas secundarias.

## Scripts disponibles

| Nombre                  | Descripción                                      | Archivo                  |
|-------------------------|--------------------------------------------------|--------------------------|
| ESP & X-Ray            | Renderizado visual mejorado de entidades         | `esp-xray.user.js`      |
| Rainbow ESP            | Efecto visual multicolor en entidades            | `rainbow-esp.user.js`   |
| Head ESP               | Destaque visual de cabezas                       | `head-esp.user.js`       |
| No Recoil              | Reducción experimental de retroceso              | `no-recoil.user.js`     |
| Sniper Speed           | Ajuste de movilidad con armas pesadas            | `sniper-speed.user.js`  |
| Better Text            | Mejora de legibilidad de texto en pantalla        | `better-text.user.js`   |

## Instalación

1. Instala la extensión [Tampermonkey](https://www.tampermonkey.net/) en tu navegador (Chrome, Firefox, Edge, etc.).
2. Ve al archivo deseado en este repositorio.
3. Haz clic en el archivo → **Raw** → Selecciona todo el código (Ctrl+A) → Copiar (Ctrl+C).
4. En Tampermonkey:
   - Clic en el icono → "Crear un nuevo script".
   - Borra el contenido predeterminado.
   - Pega el código copiado.
   - Guarda (Ctrl+S).
5. Abre [voxiom.io](https://voxiom.io) y prueba.

> Tip: Puedes tener varios scripts activos al mismo tiempo (pero con cuidado, algunos pueden interferir).
